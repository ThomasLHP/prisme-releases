# Prisme — téléchargements

Ce dépôt ne contient **que les téléchargements** de Prisme PC. Le **code source reste privé**.

## Installer / mettre à jour

1. Télécharge **`Prisme-PC.zip`** depuis la [dernière version](https://github.com/ThomasLHP/prisme-releases/releases/latest).
2. **Décompresse** le zip (clic droit → Extraire tout).
3. Ouvre le dossier **Prisme PC** et lance **`Prisme PC.exe`**.
4. Windows peut afficher un avertissement (appli non signée) → *Informations complémentaires* → *Exécuter quand même*.

Tes données restent sur ton PC, dans `C:\Users\<toi>\.prisme\`.

> Les archives « Source code (zip/tar.gz) » attachées automatiquement par GitHub ne contiennent que ce README, pas le code de l'application.
